# Unity RPG Game Project

## Project Overview
This is a Unity RPG game, designed to immerse players into a simple world with engaging gameplay mechanics and captivating storylines.

## Features
- Open-world exploration
- Character customization
- Quests and missions
- Inventory system
- Combat mechanics
- NPC interactions
- Save/load game functionality

## Technical Details
- **Game Engine**: Unity 2021.3.0f1
- **Language**: C#
- **Graphics**: 3D models and textures created using Blender and Adobe Photoshop.
- **Version Control**: Git for source code management.

## Project Structure
- **Assets/**: Contains all game assets (models, textures, audio).
- **Scripts/**: C# scripts that handle game logic.
- **Scenes/**: Unity scenes that represent different levels and environments.
- **Prefabs/**: Reusable game objects with pre-configured settings.
- **Documentation/**: Project documentation and design documents.

## How to Run It
1. Clone the repository:
   ```bash
   git clone https://github.com/Liu-is-here/the-knight.git
   ```
2. Open the project in Unity.
3. Ensure all assets are imported correctly.
4. Hit the 'Play' button in the Unity editor to start the game.
